# 🚀 PromoHub — Free Automated Promotional Website

A 100% free, automated promotional website built on GitHub Pages + GitHub Actions.

## Features
- ✅ Auto-publishes new content every 4 hours
- ✅ Pulls from any RSS feed (deals, products, news, etc.)
- ✅ Optional AI rewriting (free Google Gemini)
- ✅ Email capture for newsletter
- ✅ Ad slots ready (AdSense-compatible)
- ✅ Affiliate link support with disclosure
- ✅ Mobile responsive
- ✅ $0/month to operate

## Setup
1. Enable GitHub Pages: Settings → Pages → main branch
2. (Optional) Add `GEMINI_API_KEY` secret for AI rewrites
3. (Optional) Replace `https://formspree.io/f/PLACEHOLDER` in `index.html` with your form endpoint
4. Edit `FEEDS` array in `scripts/generate.js` to choose your niche
